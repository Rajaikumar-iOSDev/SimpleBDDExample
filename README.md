# SimpleBDDExample

This is an example of basic implementation of BDD(Given, When & Then) in swift. 

UI test cases built on top of unit test cases. Seperate test plans created for UI & unit test cases.

## Tech stack

UIKit, XCTest, Swift & Xcode.
